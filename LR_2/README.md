# Decorator Pattern – Data Formatter

## Описание проекта

Проект реализует паттерн **Декоратор** для форматирования данных в различные форматы:

* JSON
* YAML
* CSV

Система построена на основе интерфейса с использованием `ABC` и `@abstractmethod`.
Форматтеры реализуют единый интерфейс и позволяют преобразовывать данные в различные текстовые представления.

Проект написан на языке **Python** с соблюдением стандартов оформления и документирования кода.

# Архитектура

В проекте используется **паттерн "Декоратор"**, который позволяет расширять функциональность форматирования данных без изменения базового интерфейса.

Основные компоненты:

### IFormat (base.py)

Абстрактный интерфейс форматирования данных.

```python
class IFormat(ABC):

    @abstractmethod
    def format(self, data: dict) -> str:
        pass
```

Все конкретные форматтеры реализуют данный интерфейс.

---

### Форматтеры

Реализованы следующие классы:

* `JSONFormatter`
* `YAMLFormatter`
* `CSVFormatter`

Каждый класс реализует метод:

```python
format(data: dict) -> str
```

который преобразует словарь в строковое представление выбранного формата.

---

# Пример использования

```python
from formatters_app.formatters.json import JSONFormatter

data = {
    "USD": 1.0,
    "EUR": 0.9,
    "RUB": 90
}

formatter = JSONFormatter()
result = formatter.format(data)

print(result)
```
