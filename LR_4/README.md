**Фасад**— это структурный паттерн проектирования, который предоставляет простой интерфейс к сложной системе классов, библиотеке или фреймворку.

# Постановка задачи
Разработать систему умного дома, состоящую из подсистем: свет, регулирование температуры, музыка с таймером, будильник, используя паттерн фасад. Также сделать команду "режим сна" которая: выключает свет, устанавливает температуру 20 градусов, включает расслабляющую музыку для сна на 30 минут, ставит будильник на время.

**Подсистемы**:
- `LightSystem`
- `TemperatureSystem`
- `MusicSystem`
- `AlarmSystem`

**Фасад**:
- `SmartHomeFacade`
